## Расшифровка варианта
- **ИБ** — бесконечный источник  
- **ИЗ1** — пуассоновский закон генерации  
- **ПЗ1** — экспоненциальное обслуживание  
- **Д10З1** — буферизация «по кольцу»  
- **Д10О3** — отказ: «самая старая в буфере»  
- **Д2П1** — выбор прибора: по приоритету номера  
- **Д2Б3** — выбор заявки из буфера: «по кольцу»  
- **ОР2** — автоматический режим: графики (в CSV)  
- **ОД3** — пошаговый режим: временные диаграммы + текущее состояние
